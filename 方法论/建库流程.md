# 历史/社科 LLM Wiki 建库流程

> 从零开始：采集→分类→编译→建链→验证→发布

---

## 第一步：确定主题与分类体系

```yaml
分类示例（二战库）:
  - 战役: 重大战役、军事行动
  - 人物: 各方领袖、将领
  - 战场: 主要战区与战略方向
  - 部队编制: 各大国军事编制
  - 装备与技术: 武器装备、军事技术
  - 概念与政策: 战略方针、条约、审判
  - 时间线: 年表与大事记
```

## 第二步：采集 Wikipedia 源文件

```bash
# 搜索确认页面名
opencli wikipedia search "斯大林格勒战役" --lang zh --limit 5

# 获取全文
opencli wikipedia page "斯大林格勒战役" --lang zh -f md > raw/斯大林格勒战役.md

# 清洗 CLI 垃圾
sed -i '/Update available:/d; /Run: npm install/d; /^url: /d' raw/*.md

# 添加来源行
echo "" >> raw/斯大林格勒战役.md
echo "> 来源：https://zh.wikipedia.org/wiki/斯大林格勒战役" >> raw/斯大林格勒战役.md
```

## 第三步：创建指令文件

创建 `指令/` 目录下的 5 个文件：
- `00_总纲.md` — 系统概述与架构说明
- `01_分类规则.md` — 分类表与收录范围
- `02_链接规范.md` — 双向链接格式
- `03_索引页模板.md` — 索引页格式
- `04_HTML生成模板.md` — HTML 输出样式

## 第四步：编写编译脚本

```python
# wiki_init.py 核心功能：
# 1. 扫描 raw/ 按分类入库

# 3. 追加同分类关联文档
```
